# Photobooth App

A photobooth app that captures a 4-photo strip with a built-in countdown, lets you pick a border color, and
add stickers before downloading the final strip as a PNG.

## How to run

1. Clone the repo
2. Run `npm install`
3. Run `npm run dev`
4. Open http://localhost:5173

## Package for macOS (Electron)

1. Build the frontend with `npm run build`
2. Create the macOS DMG with `npm run dist`
3. Find the installer in `release/`
